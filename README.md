# 📊 Análisis de Abandono de Empleados (Employee Attrition Analysis)

## 📌 Descripción

Este proyecto presenta un análisis integral sobre la rotación de empleados (Employee Attrition), con el objetivo de identificar los principales factores asociados al abandono laboral y generar información útil para apoyar la toma de decisiones dentro del área de Recursos Humanos.

El desarrollo sigue un enfoque **end-to-end**, abarcando desde la obtención y preparación de los datos hasta la construcción de dashboards ejecutivos en Power BI.

---

## 🎯 Objetivos

- Analizar el comportamiento del abandono de empleados.
- Identificar perfiles con mayor riesgo de renuncia.
- Detectar variables con mayor impacto sobre la rotación.
- Construir indicadores clave para la toma de decisiones.
- Diseñar un dashboard ejecutivo interactivo.

## 🔍 Principales Hallazgos

Algunos de los principales insights encontrados fueron:

- La tasa de abandono alcanza aproximadamente el 16%.
- Los empleados que realizan horas extra presentan una mayor proporción de abandono.
- Los primeros años dentro de la empresa concentran la mayor cantidad de renuncias.
- Research & Development registra el mayor número absoluto de bajas debido a su tamaño.
- Los empleados jóvenes muestran una mayor rotación respecto de otros grupos etarios.
